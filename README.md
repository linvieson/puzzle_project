PUZZLE PROJECT


About the program

The puzzle program is used for validating if the board is ready for the game or not. The board is 9x9 cells,
which are either blank or contain numbers.


What this program does?

This program checks if the board is ready for the game. To get the True result the board must follow all the next rules.
Cells of the same color must not contain similar numbers. Cells in one line must not contain similar numbers.
Cells in one column must not contain similar numbers. If at least one rule is not followed, the output result
is False.


Instruction on how to use the program

Call the validate_board() function with board as the paramether in it. Start the program and wait for the result.


Results of the work

A program that validates if the board is ready for the game or not.
